「JavaScript」JavaScript作用域

JaveScript 是Web前端开发的最重要的语言，在JavaScript中作用域是一个重要的基础知识点。

在JavaScript中共有三种作用域，分别是全局作用域、函数左右域和块级作用域。

本文会先对三种作用域做介绍，之后举一个常见的例子做整体介绍。
